# 第 2 章：组合逻辑设计

> DDCA Chapter 2: Combinational Logic Design

## 本章目标

- 掌握布尔代数的基本定律和化简技巧
- 能够用卡诺图（K-Map）化简逻辑表达式
- 理解组合逻辑的时序参数（传播延迟、最小延迟）
- 能够从真值表推导出最简逻辑电路

## 实验

| 编号 | 实验 | 说明 |
|------|------|------|
| E01 | 布尔代数化简验证 | 用 Verilog 验证布尔恒等式 |
| E02 | 卡诺图化简 | 给定真值表，手算化简并用仿真验证 |
| E03 | 传播延迟测量 | 测量不同级联深度的门延迟 |

## 与项目的关系

**M1: 基本逻辑单元库** — 建立参数化门级元件库：`gates.v`，包含 AND/NAND/OR/NOR/XOR/XNOR 的参数化模块。
