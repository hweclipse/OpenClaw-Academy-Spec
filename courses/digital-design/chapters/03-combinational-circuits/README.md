# 第 3 章：组合逻辑电路

> DDCA Chapter 2 & Chapter 5: Combinational Building Blocks

## 本章目标

- 理解并设计译码器（Decoder）和编码器（Encoder）
- 理解并设计多路选择器（Multiplexer）和多路分配器
- 掌握组合逻辑模块的级联与扩展
- 理解优先级编码器的工作原理

## 实验

| 编号 | 实验 | 说明 |
|------|------|------|
| E01 | 3-to-8 译码器 | 设计带使能端的 3-8 译码器 |
| E02 | 8-to-3 优先级编码器 | 设计级联优先级编码器 |
| E03 | 参数化多路选择器 | 设计 N:1 MUX，支持任意数据位宽 |

## 与项目的关系

扩展现有的门级元件库，增加译码器和多路选择器。这些模块将在第 7 章 ALU 中使用。
