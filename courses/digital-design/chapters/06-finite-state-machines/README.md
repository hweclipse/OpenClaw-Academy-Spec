# 第 6 章：有限状态机

> DDCA Chapter 3: Finite State Machines

## 本章目标

- 理解 Moore 机和 Mealy 机的区别
- 掌握状态图绘制和状态编码
- 能够从规格说明书推导出 FSM 状态转移图
- 掌握 FSM 的 Verilog 实现（三段式写法）
- 理解状态编码对电路面积和速度的影响

## 实验

| 编号 | 实验 | 说明 |
|------|------|------|
| E01 | 序列检测器 (Mealy) | 设计 "1101" 序列检测器 |
| E02 | 交通灯控制器 (Moore) | 设计十字路口交通灯 FSM |
| E03 | 状态编码对比 | 对比 Binary / Gray / One-Hot 编码的资源差异 |

## 与项目的关系

FSM 是第 9 章外设控制器的核心技术。本章的序列检测器和交通灯控制器为更复杂的协议控制器做准备。
