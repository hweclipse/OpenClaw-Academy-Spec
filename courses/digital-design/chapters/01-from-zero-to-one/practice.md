## 练习 2：十进制 → 二进制

将以下十进制数转换为 8 位二进制（不足 8 位时高位补 0）：

| 十进制 | 8 位二进制 |
|--------|-----------|
| 42 | |
| 100 | |
| 7 | |
| 200 | |

### 答案

| 十进制 | 8 位二进制 |
|--------|-----------|
| 42 | `0b0010_1010` |
| 100 | `0b0110_0100` |
| 7 | `0b0000_0111` |
| 200 | `0b1100_1000` |

**计算过程**（以 42 为例）：
```
42 ÷ 2 = 21 余 0 (LSB)
21 ÷ 2 = 10 余 1
10 ÷ 2 = 5  余 0
5 ÷ 2  = 2  余 1
2 ÷ 2  = 1  余 0
1 ÷ 2  = 0  余 1 (MSB)

结果：101010₂ → 补 2 位 → 00101010₂
```

## 练习 5：CMOS 门分析

对于 CMOS NAND 门：

(a) 当 A=0, B=1 时，哪个（哪些）晶体管导通？
(b) 此时输出 Y 的值是多少？为什么？

### 答案

(a) 当 A=0, B=1 时：
- PMOS_A（Gate=0）→ **导通**
- PMOS_B（Gate=1）→ 截止
- NMOS_A（Gate=0）→ 截止
- NMOS_B（Gate=1）→ 导通

但由于 NMOS_A 和 NMOS_B 是串联的（PDN），必须两者同时导通才能下拉。NMOS_A 截止 → PDN 断开。

PUN 中 PMOS_A 导通 → Y 连接到 VDD。

(b) Y = **1**。因为上拉网络（PUN）提供了 VDD 到 Y 的通路，而下拉网络（PDN）断开。

## 练习 6：用 NAND 构建其他门

仅用 2 输入 NAND 门，如何实现 NOT 门？画出连接方式。

### 答案

将 NAND 的两个输入端连接在一起：

```
A ─┬─┐
   │ NAND ├── Y = A NAND A = NOT A
A ─┴─┘
```

验证：
- A=0 → 0 NAND 0 = 1 ✓
- A=1 → 1 NAND 1 = 0 ✓
