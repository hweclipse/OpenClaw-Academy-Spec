# 第 5 章：时序逻辑设计

> DDCA Chapter 3: Sequential Logic Design

## 本章目标

- 理解锁存器（Latch）和触发器（Flip-Flop）的区别
- 掌握 D 触发器、D 锁存器的行为
- 理解同步时序电路设计方法
- 理解建立时间（setup time）和保持时间（hold time）
- 掌握同步复位 vs 异步复位的设计

## 实验

| 编号 | 实验 | 说明 |
|------|------|------|
| E01 | D 锁存器与 D 触发器 | 设计并仿真比较两者的时序行为 |
| E02 | 参数化寄存器 | 设计 N 位带使能和复位的寄存器 |
| E03 | 同步 vs 异步复位 | 实验比较两种复位方式的时序差异 |

## 与项目的关系

**M2: 寄存器与计数器** — 参数化寄存器、移位寄存器、通用计数器。这些是第 7 章 ALU 数据通路的基础。
